ProfitSleuth
Resale Intelligence — Powered by Gemini

ProfitSleuth is an AI-powered utility designed to eliminate the manual drudgery of reselling. By automating data entry and market analysis, ProfitSleuth empowers resellers to move faster, make data-backed acquisition decisions on the spot, and increase their sourcing velocity.

The Problem
Manual lookups in liquidation and return stores are a significant bottleneck. Standing in aisles typing titles into multiple browsers, guessing shipping fees, and calculating platform cuts turns what should be a profitable "hustle" into tedious data entry.

The Solution
A clean, 30-second workflow. ProfitSleuth allows users to:

Snapshot items: Instantly fetch real market values.

Calculate margins: Automatically account for dynamic fees and shipping costs.

Generate listings: Produce professional product descriptions and SEO tags in seconds.

Tech Stack
AI: Gemini Vision

Frameworks: React Native, Expo Router

Backend: Node API

Hosting: GitHub Pages

Impact
ProfitSleuth transforms blind guesswork into data-backed acquisition, enabling a 10x increase in sourcing velocity.
